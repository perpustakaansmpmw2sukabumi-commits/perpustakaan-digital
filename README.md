# 📚 Perpustakaan Digital

Web perpustakaan digital modern yang di-host di GitHub Pages.

## ✨ Fitur
- 🔍 Pencarian real-time
- 🏷️ Filter kategori
- 📱 Responsive design
- 💫 Animasi modern
- 📖 Modal detail buku
- 🎨 Glassmorphism UI

## 🚀 Deploy ke GitHub Pages
1. Fork atau buat repository baru
2. Upload semua file
3. Settings → Pages → Deploy from branch: `main`
4. Akses: `https://username.github.io/nama-repo`

## 📝 Cara Kustomisasi
1. Edit `books.json` untuk tambah buku
2. Ganti cover image URL
3. Update PDF links (gunakan Project Gutenberg untuk legal PDFs)
